# AGENTS.md

## 프로젝트 작업 규칙

1. 이미지는 무조건 imagegen 스킬을 사용해서 생성한다. 코드나 벡터를 통한 생성은 차단한다.
2. UI는 무조건 canvas로. canvas로 불가능한 것만 런타임으로 구현한다.

## 새 세션 시작 규칙

새 컴퓨터, 새 Codex 서브 에이전트, 새 세션에서 작업을 시작하면 먼저 [START-HERE.md](START-HERE.md)를 읽는다.

작업 목표는 항상 현재 합의된 첫 수직 슬라이스를 기준으로 잡는다. 큰 아이디어를 추가하더라도 먼저 문서화하고, 구현은 `준비 -> 탐사 -> 수색/전리품 획득 -> 귀환 -> 정리/제작 -> 다음 목표` 루프에 도움이 되는 범위로 제한한다.
