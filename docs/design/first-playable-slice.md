# Project D 첫 플레이 가능 수직 슬라이스

## 목적

이 문서는 Project D의 첫 구현 목표를 고정하기 위한 설계 문서다.

첫 목표는 완성된 프롤로그 전체가 아니라, Project D의 핵심 재미인 `준비 -> 원정 -> 수색/루팅 -> 귀환 -> 정산 -> 다음 목표`가 작동하는 15분짜리 플레이 가능 루프를 만드는 것이다.

## 목표 플레이 시간

- 최소: 15분
- 확장: 30분

## 플레이어 경험 목표

플레이어는 첫 수직 슬라이스에서 다음 감각을 얻어야 한다.

- 오두막은 안전한 거점이다.
- 숲은 원래 익숙했지만 지금은 이상해졌다.
- 원정 전 준비가 생존에 영향을 준다.
- 모든 아이템을 가져갈 수는 없다.
- 살아서 돌아와야 전리품이 확정된다.
- 귀환 후 다음 선택이 생긴다.

## 확정된 초반 흐름

### 1. 메인 메뉴

포함:

- 새 게임
- 이어하기
- 환경설정
- 종료

환경설정 최소 범위:

- 해상도
- 그래픽 품질
- 전체화면
- 마스터 볼륨
- 효과음 볼륨
- 마우스 감도

### 2. 조작 가능한 마법 폭주 프롤로그

목표:

- 디아나가 마력을 잃은 사건을 플레이어가 수동 컷씬으로만 보지 않게 한다.

구성:

- 연구실에서 룬 또는 마도구를 조작한다.
- 플레이어가 간단한 입력을 수행한다.
- 안정화에 실패하고 마법이 폭주한다.
- 디아나는 마력을 잃는다.

주의:

- 길게 만들지 않는다.
- 고품질 컷씬은 나중으로 미룬다.
- 첫 버전에서는 검은 화면, 자막, 임시 연출로 대체 가능하다.

### 3. 움브라데 오두막 정착

목표:

- 안전한 공간에서 기본 조작, 채집, 제작, 수리를 배운다.

필수 행동:

- 채집 1회
- 인벤토리 확인 1회
- 제작 1회
- 오두막 수리 1회
- 보관함 사용 1회

주의:

- 오두막 전체를 완성시키지 않는다.
- 반복 채집 노동을 요구하지 않는다.
- "비를 피할 정도로 고쳤다" 수준이면 충분하다.

### 4. 분위기 전환

목표:

- 같은 숲이 평화로운 생활 공간에서 위험한 원정지로 변했음을 보여준다.

연출:

- 안개 증가
- 새소리 감소
- 낮은 바람 소리
- 먼 비명 또는 이상한 동물 울음
- 채집터 일부 오염
- 발자국, 피 묻은 물, 썩은 냄새

### 5. 오염된 시체 발견

목표:

- 좀비 사태가 일반 언데드나 자연 부패가 아니라는 점을 알린다.

구성:

- 디아나가 오염된 시체를 조사한다.
- 부패 오염 게이지가 소량 증가할 수 있다.
- 디아나는 "언데드가 아니라 살아있는 신체가 부패한 흔적"임을 분석한다.
- 오두막으로 돌아가 조사 준비를 하기로 결정한다.

### 6. 첫 원정 준비

목표:

- 타르코프식 무장 점검을 Project D 세계관에 맞게 번역한다.

UI:

- 원정 준비 체크리스트

체크 항목:

- 무기
- 회복품
- 가방 여유
- 조사 목표

첫 버전 제한:

- 무기 선택지는 1~2개
- 회복품 1종
- 가방 1종
- 목표 1개

### 7. 첫 숲 원정

목표:

- 전투보다 익스트랙션 사고방식을 가르친다.

핵심 감각:

- 들어간다.
- 이상한 것을 본다.
- 조금 줍는다.
- 위험해진다.
- 더 욕심낼지 돌아갈지 고민한다.
- 살아서 귀환한다.

맵 구성:

- 오두막 앞마당: 안전지대, 귀환 지점
- 익숙한 채집 숲길: 이전 튜토리얼 공간의 변화 확인
- 개울가: 발자국, 피 묻은 물, 감염 동물 흔적
- 무너진 수레/난민 흔적: 루팅 포인트
- 부패한 시체 발견지: 핵심 조사 목표
- 낡은 감시초소: 선택 보상과 귀환로 확인
- 마법 지뢰 경고 구역 가장자리: 아직 진입 금지, 세계관 암시

적 배치:

- 비전투 동물 또는 흔적
- 이상 행동을 보이는 감염 동물은 첫 버전에서 흔적만 가능
- 약한 감염체 1마리
- 이후 약한 감염체 2마리까지 가능
- 무법자는 직접 전투 없이 흔적으로 암시

### 8. 귀환과 정산

목표:

- 살아서 돌아온 물건만 확정된다는 장르 규칙을 알려준다.

정산 화면 최소 항목:

- 가져온 전리품
- 보관하기
- 제작에 사용
- 조사 과제 확인

첫 버전에서는 판매, 분해, 상인 거래, 복잡한 연구 트리는 열지 않는다.

### 9. 본 게임 루프 진입

귀환 후 첫 조사 과제가 열린다.

예시:

- `부패 샘플 분석하기`
- `오두막 작업대에서 정화 도구 만들기`
- `다음 원정에서 오염 근원 찾기`

## 실패 처리

첫 원정에서 실패해도 완전 게임오버로 처리하지 않는다.

가능한 처리:

- 디아나가 간신히 오두막으로 돌아온다.
- 일부 루팅품을 잃는다.
- 하루가 지난다.
- 치료 자원이 소모된다.
- 부패 오염 치료가 필요해진다.

실패가 알려줘야 하는 것:

- 전투는 피할 수도 있다.
- 오염 시체와 감염체는 준비 없이 만지면 위험하다.
- 모든 아이템을 가져갈 수 없다.
- 귀환로와 랜드마크 확인이 중요하다.

## MVP 포함 범위

- 메인 메뉴와 기본 옵션
- 짧은 프롤로그 시퀀스
- 오두막 거점 1개
- 채집/루팅
- 간단 제작/수리
- 원정 준비 UI
- 작은 숲 맵 1개
- 약한 감염체 1종
- 체력과 회복
- 부패 오염 또는 출혈 중 하나
- 귀환 지점
- 귀환 정산
- 첫 조사 과제

## 나중으로 미룰 범위

- 자유 건축
- 풀 보이스 컷씬
- 대형 오픈월드 숲
- 복잡한 생태계 AI
- 인간형 무법자 전투 AI
- 다중 조건부 귀환로
- 복잡한 상인 경제
- 깊은 장비 내구도/무게/부상 시스템
- 대규모 거점 확장 트리

## 성공 기준

첫 수직 슬라이스는 다음이 가능하면 성공이다.

- 플레이어가 오두막에서 준비한다.
- 숲에 나간다.
- 오염 흔적을 조사한다.
- 아이템을 얻는다.
- 약한 위협을 만나 판단한다.
- 살아서 귀환한다.
- 전리품을 분류한다.
- 다음 목표가 생긴다.

이 루프가 재미있으면 이후 시스템을 확장한다.
