# Project D 첫 플레이 가능 수직 슬라이스

## 목적

이 문서는 Project D의 첫 구현 목표를 고정하기 위한 설계 문서다.

첫 목표는 완성된 프롤로그 전체가 아니라, Project D의 핵심 재미인 `준비 -> 탐사 -> 수색/전리품 획득 -> 귀환 -> 정리 -> 제작 -> 다음 목표`가 작동하는 플레이 가능 루프를 만드는 것이다.

## 목표 플레이 시간

- 목표: 20분~30분

## 탐사 제한시간 정책

탐사에는 맵별 제한시간이 있다.

초기 개발 단계에서 움브라데 숲의 탐사 제한시간은 25분이다.

세계관 설정상 탐사 제한시간은 감염 원인이 되는 바이러스가 공기 중에 퍼진 지역의 안전 임계점이다. 임계점을 넘기면 디아나는 더 이상 정상적인 탐사를 지속할 수 없다.

맵별 제한시간이 다른 이유는 지역마다 바이러스 농도와 확산 속도가 다르기 때문이다.

시간 압박은 별도 단계 연출 없이 타이머로 전달한다.

- `O`를 누르면 타르코프처럼 남은 시간이 표시된다.
- `O`를 더블 탭하면 귀환 지점 목록까지 표시된다.
- 인벤토리/메뉴 화면에는 별도 타이머를 표시하지 않는다.
- 타이머가 0이 되면 타임오버 실패 처리로 넘어간다.
- 타이머 시작 시점은 실제 탐사 지역에서 플레이어 조작이 가능한 시점이다.

타임오버는 사망이나 실종 처리 대신, 디아나가 위독해져 모든 짐을 내던지고 오두막으로 급하게 돌아오는 것으로 처리한다.

## 플레이어 경험 목표

플레이어는 첫 수직 슬라이스에서 다음 감각을 얻어야 한다.

- 오두막은 안전한 거점이다.
- 숲은 원래 익숙했지만 지금은 이상해졌다.
- 탐사 전 준비가 생존에 영향을 준다.
- 모든 아이템을 가져갈 수는 없다.
- 살아서 돌아와야 전리품이 확정된다.
- 귀환 후 다음 선택이 생긴다.

## 확정된 초반 흐름

### 1. 메인 메뉴

포함:

- 게임시작
- 환경설정
- 게임종료

저장 정책:

- 게임은 하나의 저장 파일만 사용한다.
- `새 게임`과 `이어하기`를 분리하지 않는다.
- `게임시작`을 누르면 바로 게임을 시작하고 세이브 파일을 불러온다.
- 진행 초기화는 `환경설정 -> 기타 -> 세계 초기화`에서 처리한다.
- 오두막 내부에서는 실시간 저장한다.
- 탐사 중에는 저장하지 않고, 탐사가 끝난 뒤 저장한다.

환경설정 최소 범위:

- 해상도
- 그래픽 품질
- 전체화면
- 마스터 볼륨
- 효과음 볼륨
- 마우스 감도

### 2. 마법 폭주 프롤로그

목표:

- 디아나가 마력을 잃은 사건을 플레이어가 수동 컷씬으로만 보지 않게 한다.

구성:

- 연구실에서 마법 안정화 미니게임을 진행한다.
- 미니게임 이후 거대한 마법 폭풍이 아카데미에서 발생한다.
- 애니메이션 컷씬이 재생된다.
- 디아나는 마력을 잃는다.
- 추방 과정은 애니메이션 컷씬으로 표현한다.

주의:

- 길게 만들지 않는다.
- 고품질 컷씬은 나중으로 미룬다.
- 임시 버전에서는 미니게임과 컷씬의 연출 품질을 낮춰도 되지만, 구조는 유지한다.

### 3. 움브라데 오두막 정착

목표:

- 안전한 공간에서 기본 조작, 채집, 제작, 수리를 배운다.

필수 행동:

- 채집 1회
- 인벤토리 확인 1회
- 제작 1회
- 창고에 있는 재료를 이용한 오두막 수리 상호작용 1회
- 보관함 사용 1회

주의:

- 오두막 전체를 완성시키지 않는다.
- 반복 채집 노동을 요구하지 않는다.
- 첫 튜토리얼 분량은 20분~30분을 목표로 한다.
- 오두막은 초반에는 `오두막`으로 부르고, 스토리 중반에 마을 형태로 확장되면 플레이어가 마을 이름을 짓는다.
- 오두막은 완전히 버려진 곳이므로 초반에는 연구 책상이나 망가진 마도구를 배치하지 않는다.

### 4. 분위기 전환

목표:

- 같은 숲이 평화로운 생활 공간에서 위험한 탐사지로 변했음을 보여준다.

연출:

- 안개 증가
- 새소리 감소
- 낮은 바람 소리
- 먼 비명 또는 이상한 동물 울음
- 채집터 일부 오염
- 발자국, 피 묻은 물, 썩은 냄새

### 5. 오염된 시체 발견

목표:

- 좀비 사태가 일반 언데드나 자연 부패가 아니라는 점을 알린다.

구성:

- 디아나가 오염된 시체를 조사한다.
- 공기 중 바이러스와 감염 위험을 암시한다.
- 디아나는 "언데드가 아니라 살아있는 신체가 부패한 흔적"임을 분석한다.
- 오두막으로 돌아가 조사 준비를 하기로 결정한다.

### 6. 첫 탐사 준비

목표:

- 타르코프식 무장 점검을 Project D 세계관에 맞게 번역한다.

UI:

- 탐사 준비 체크리스트

체크 항목:

- 무기
- 회복품
- 가방 여유
- 조사 목표

첫 버전 제한:

- 무기 선택지는 1~2개
- 회복품 1종
- 가방 1종
- 목표 1개

### 7. 첫 숲 탐사

목표:

- 전투보다 익스트랙션 사고방식을 가르친다.

핵심 감각:

- 들어간다.
- 이상한 것을 본다.
- 조금 줍는다.
- 위험해진다.
- 더 욕심낼지 돌아갈지 고민한다.
- 숲의 봉우리에 있는 오래된 감시탑에 올라 주변을 확인한다.
- 살아서 귀환한다.

맵 구성:

- 오두막 앞마당: 안전지대, 귀환 지점
- 익숙한 채집 숲길: 이전 튜토리얼 공간의 변화 확인
- 개울가: 발자국, 피 묻은 물, 감염 동물 흔적
- 무너진 수레/난민 흔적: 수색 포인트
- 부패한 시체 발견지: 오염 징후 확인
- 오래된 감시탑: 핵심 탐사 목표, 주변 확인
- 마법 지뢰 경고 구역 가장자리: 아직 진입 금지, 세계관 암시

적 배치:

- 비전투 동물 또는 흔적
- 감염 동물은 시체로만 등장
- 첫 적은 감염체
- 무법자는 도망가다 감염체에게 당하는 연출로 등장
- 선택 보상은 넣지 않고 필수 보상만 둔다.

### 8. 귀환과 정산

목표:

- 살아서 돌아온 물건만 확정된다는 장르 규칙을 알려준다.

정산 화면 최소 항목:

- 가져온 전리품
- 보관하기
- 제작에 사용
- 의뢰 확인

첫 버전에서 제외할 시스템은 아직 미정이다.

### 9. 실패와 보험 회수

목표:

- 사망/타임오버의 손실은 강하게 느끼게 하되, 싱글 플레이어 초반 진행을 끊지 않는다.

실패 처리:

- 탐사 도중 치명상을 입으면 디아나는 모든 짐을 내던지고 오두막으로 급하게 돌아온다.
- 탐사 제한시간을 넘기면 디아나가 위독해지고, 모든 짐을 내던지고 오두막으로 급하게 돌아온다.
- 손실 범위는 타르코프의 secure container 내부 물건을 제외한 전부와 같은 구조를 따른다. Project D의 해당 보관 수단 명칭은 아직 미정이다.
- 기본 무기, 기본 도구, 의뢰 핵심 아이템도 손실될 수 있다.
- 실패 후 페널티는 아이템 유실이다.
- 첫 튜토리얼 탐사에서도 아이템 유실 페널티를 적용한다.

보험 회수 설정:

- Project D에는 보험업자 NPC가 존재한다.
- 보험업자는 초반 의뢰 라인에서 등장한다.
- 보험업자에게 직접 가서 보험을 들 수 있고, 탐사 전 보험드는 단계도 있다.
- 보험 대상은 소모품이 아닌 모든 것이다.
- 보험업자는 마법생물 하수인을 부려 탐사 중 잃어버린 아이템을 회수하게 한다.
- 마법생물 하수인의 형태는 아직 미정이다.
- 회수 가능 여부와 반환 시간은 아이템 위치, 지역 위험도, 바이러스 농도, 보험 조건에 따라 달라진다.
- 회수 성공률은 기본 100%이지만, 일부 맵에서는 확률 감소 또는 회수 불가가 있을 수 있다.
- 첫 수직 슬라이스에서는 보험 시스템을 완성하지 않고, 세계관 설정과 후속 시스템 후보로 문서화한다.

### 10. 본 게임 루프 진입

귀환 후 첫 의뢰가 열린다.

예시:

- `부패 샘플 분석하기`
- `오두막 작업대에서 정화 도구 만들기`
- `다음 탐사에서 오염 근원 찾기`

## 실패 처리

첫 탐사에서 실패해도 완전 게임오버로 처리하지 않는다.

가능한 처리:

- 디아나가 간신히 오두막으로 돌아온다.
- 일부 전리품을 잃는다.

실패가 알려줘야 하는 것:

- 전투는 피할 수도 있다.
- 오염 시체와 감염체는 준비 없이 만지면 위험하다.
- 모든 아이템을 가져갈 수 없다.
- 귀환로와 랜드마크 확인이 중요하다.

## 첫 버전 포함 범위

- 메인 메뉴와 기본 옵션
- 짧은 프롤로그 시퀀스
- 오두막 거점 1개
- 채집/수색
- 간단 제작/수리
- 탐사 준비 UI
- 작은 숲 맵 1개
- 약한 감염체 1종
- 체력과 회복
- 귀환 지점
- 귀환 정산
- 첫 의뢰
- 움브라데 숲 25분 제한시간
- 타임오버 시 짐을 버리고 오두막으로 귀환하는 실패 처리
- 아이템 50개 이하
- 제작 레시피 19개 이하
- 나무상자, 창고상자, 시체, 파밍 포인트 내 랜덤 스폰

## 성공 기준

첫 수직 슬라이스는 다음이 가능하면 성공이다.

- 플레이어가 오두막에서 준비한다.
- 숲에 나간다.
- 오염 흔적을 조사한다.
- 아이템을 얻는다.
- 약한 위협을 만나 판단한다.
- 살아서 귀환한다.
- 전리품을 분류한다.
- 다음 목표가 생긴다.

이 루프가 재미있으면 이후 시스템을 확장한다.
