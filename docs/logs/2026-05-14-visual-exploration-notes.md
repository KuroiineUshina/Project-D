# 2026-05-14 시각 콘셉트 탐색 기록

이 문서는 다른 컴퓨터나 새 Codex 세션에서 Project D 시각 작업을 이어가기 위한 메모다.
생성된 이미지 파일은 대부분 불합격 또는 실험 산출물이므로 GitHub에 올리지 않는다.

## 이번 세션에서 확인한 것

- UI 이미지 생성 시도가 여러 차례 있었지만, 사용자가 최종적으로 전부 불합격 처리했다.
- 가장 큰 문제는 UI가 너무 많은 정보를 동시에 보여준다는 점이었다.
- Project D의 탐사 중 UI는 Tarkov의 레이드 HUD처럼 정보를 거의 주지 않는 방향이어야 한다.
- Tarkov 참고 시 반드시 `레이드 중 HUD`와 `레이드 전후 준비/스태시 UI`를 구분해야 한다.
- 레이드 전후 화면은 정보량이 많아도 되지만, 탐사 중에는 미니맵, 상시 퀘스트 추적, 적 표시, 상시 타이머, 전체 인벤토리 표시를 피해야 한다.
- 타이머는 `O`를 누르고 있을 때만 표시한다.
- 귀환 지점 목록은 `O` 더블탭 때만 표시한다.
- 상호작용 프롬프트는 실제 상호작용 가능한 대상 앞에서만 짧게 표시한다.

## UI 방향 메모

탐사 중 기본 화면:

- HUD는 거의 비운다.
- 미니맵 없음.
- 상시 의뢰 목록 없음.
- 적 마커 없음.
- 상시 타이머 없음.
- 상태 정보는 매우 작거나, 화면 효과/캐릭터 반응/간단한 아이콘으로만 표현한다.

수색 순간:

- `F 수색` 같은 짧은 프롬프트만 우선한다.
- 수색 결과도 작은 알림이나 짧은 목록 위주로 보여준다.
- 전체 가방 그리드는 탐사 중 기본 수색 화면에 바로 띄우지 않는다.

준비/정리 단계:

- 장비, 가방, 보관함, 보험, 제작, 의뢰 화면은 정보량이 많아도 된다.
- 단, 한 화면에 모든 기능을 밀어 넣지 말고 현재 선택한 기능 중심으로 좁힌다.

## 디아나 캐릭터 초안 메모

선택된 기본 정보:

- 이름: 디아나
- 종족: 엘프
- 성별: 여성
- 나이: 327세, 엘프 기준 청년
- 키/체형: 152cm, 보통의 마른 체형
- 머리: 푸른 빛이 도는 은색 긴 생머리
- 눈: 밤하늘이 담긴 듯한 검푸른 눈동자
- 출신: 엘프 국가 아우루미나스
- 직업/배경: 유실된 고대 마법을 연구하던 학자

이번에 만든 디아나 턴어라운드 초안도 사용자가 서브컬처 게임 그림체 이해가 부족하다고 판단했다.
다음 시도에서는 막연히 `서브컬처풍`이라고 쓰지 말고, 서브컬처 게임 3개 이상을 먼저 조사한 뒤 캐릭터 비율, 얼굴 조형, 선 정리, 의상 레이어, 색감, 모델링 친화성을 프롬프트에 구체적으로 반영해야 한다.

## 이미지 파일 처리 규칙

- `outputs/` 폴더는 Git 추적에서 제외한다.
- 불합격 이미지와 실험 이미지 파일은 GitHub에 올리지 않는다.
- 필요하면 다음 세션에서 로컬 폴더를 직접 참고하되, 저장소에는 결정사항과 분석 메모만 남긴다.
- 최종 채택된 이미지가 생기기 전까지 이미지는 산출물 후보일 뿐이며, 저장소의 공식 아트로 취급하지 않는다.

## 다음 세션 추천 시작점

1. 캐릭터 구조부터 다시 잡는다.
2. 디아나의 실루엣, 얼굴상, 의상 레이어, 세계관 기능을 글로 먼저 확정한다.
3. 서브컬처 게임 레퍼런스를 조사해 `그림체 분석 -> 프롬프트 -> A포즈 턴어라운드` 순서로 다시 생성한다.
4. UI 작업은 캐릭터 방향이 조금 잡힌 뒤 재개한다.
5. UI를 다시 만들 때는 탐사 중 정보 노출 규칙을 먼저 문서화한 뒤 이미지를 생성한다.
